## Built With

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- python 3.4+ (for asyncio)

### Installation

1. Download the script

2. Install dependencies

```sh
pip install python-socketio aiohttp
```

## Usage

Run app

```sh
python ./socket_server.py
```

### Socket Usage
